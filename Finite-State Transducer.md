# Finite-State Transducer

A finite automaton that, while reading an input string, emits an output string — effectively a rational relation between two languages. In automated ACL evolution, each agent's translator between its private knowledge-representation language and the shared ACL is modelled as an FST whose states/transitions can be extended via negotiation.
