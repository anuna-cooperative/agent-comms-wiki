# Vocabulary Size Penalty

A Dirichlet-process-inspired regulariser in Mordatch & Abbeel's emergent-language setup that penalises usage of rarely-seen symbols, pushing agents toward compact, reusable vocabularies and thereby encouraging compositional rather than holistic encodings.
