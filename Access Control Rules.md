# Access Control Rules

In LACIOS and related data-driven coordination languages, declarative rules that govern which agents may perceive, retrieve, or modify symbolic descriptions in a shared tuple-space. Global rules specify system-wide perception/retrieval policies; local rules let individual agents contextually hide themselves or their data, separating coordination logic from access policy.
