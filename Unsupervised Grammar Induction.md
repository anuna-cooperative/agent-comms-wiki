# Unsupervised Grammar Induction

The learning problem of inferring a grammar's productions from corpora without annotated parses — essentially, discovering the syntactic structure of a language from usage alone. Invoked in automated ACL evolution where agents must jointly infer compatible grammars for new shared constructs.
