# BDI Architecture

The Belief-Desire-Intention model of practical reasoning: an agent maintains beliefs about the world, desires (goals) it would like to achieve, and intentions (committed plans) that drive its behaviour, typically executed via a sense-deliberate-act loop over a plan library.
