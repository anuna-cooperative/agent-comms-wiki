# Self-aware Computing

A design paradigm where a system maintains an explicit runtime model of itself (capabilities, state, performance, ethics constraints) and uses it to monitor, adapt, and explain its behaviour. Concretised in trustworthy-logical-agent work via runtime verification against A-ILTL/trace-expression specifications.
