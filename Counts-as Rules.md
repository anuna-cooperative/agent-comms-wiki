# Counts-as Rules

Constitutive rules of the form "X counts as Y in context C" (Searle), used in artificial-institution frameworks to define when a brute act (sending a message) institutionally counts as a declaration that creates, updates or cancels a commitment. The mechanism by which message exchanges produce institutional facts.
