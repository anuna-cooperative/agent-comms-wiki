## TOTE Loop

Miller, Galanter and Pribram's Test-Operate-Test-Exit cycle: a basic cognitive control structure in which a system compares state to a goal (Test), acts to reduce the discrepancy (Operate), re-tests, and exits on match. Levin uses it as a primitive for building up scale-free cognition.
