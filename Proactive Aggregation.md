# Proactive Aggregation

Gossip-based aggregation style (Jelasity et al.) in which all nodes periodically exchange with random peers to continuously maintain an estimate of a global aggregate, as opposed to reactive/on-demand computation. Produces a continuously-refreshed value robust to churn.
