# ACL Design Principles

Eight criteria proposed by Kone, Shimazu and Nakajima for evaluating agent communication languages: heterogeneity, cooperation/coordination, separation (of message from transport and content), interoperability, transparency, extensibility/scalability, performance, and security. The principles accompany a four-part ACL template (message format, semantic model, interaction protocols, shared ontology/content language).
