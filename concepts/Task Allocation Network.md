# Task Allocation Network

A multi-agent organisational structure in which agents maintain weighted peer-to-peer and superior-subordinate relationships, with subtasks routed to partners chosen by trust-fused candidate selection. The substrate on which DSmT-based trust fusion and multi-agent Q-learning drive self-organisation.
