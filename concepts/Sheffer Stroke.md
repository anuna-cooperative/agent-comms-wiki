# Sheffer Stroke

The NAND connective (¬(p ∧ q)), notable because every Boolean function can be built from it alone — a single *universal* logical operation. Von Neumann analyses it as an alternative universal basic organ for his reliability construction, showing the multiplexing argument does not depend on the particular choice of elementary gate.
