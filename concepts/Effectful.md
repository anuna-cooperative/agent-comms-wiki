# Effectful

Basis Research Institute's Python library for [[Algebraic Effects]] (2024). Provides effect operations and handlers as first-class Python objects, enabling the same program to run under different interpretations. Used by [[Pact - A Choreographic Language for Agentic Ecosystems]] to implement [[Endpoint Projection]]: the choreographic program is written once, and each role's local view is obtained by installing that role's handler.
