# Possible Worlds Semantics

The semantic foundation for modal and [[Epistemic Logic|epistemic]] reasoning (Hintikka, Kripke): the meaning of "necessarily φ" or "agent *i* knows φ" is given by quantifying over a set of *possible worlds*. An agent knows φ when φ is true in all worlds it cannot distinguish from the actual one; uncertainty is modelled as the set of worlds an agent holds possible. Formalised concretely by [[Kripke Structure|Kripke structures]] and, for distributed settings, by [[Interpreted Systems]].
