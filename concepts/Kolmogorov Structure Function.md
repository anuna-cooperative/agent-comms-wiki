# Kolmogorov Structure Function

For a string x and a complexity budget k, the minimal log-cardinality of a finite set S with x in S and K(S) at most k; separates the *structural* (model) part of x from its residual *random* part. The foundation of algorithmic statistics and of the Minimum Description Length principle's model-selection guarantees.
