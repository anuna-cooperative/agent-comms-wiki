# Kripke Structure

A model for [[Epistemic Logic|modal/epistemic logic]]: a set of *possible worlds* (states) together with binary accessibility relations — one per agent — and a valuation fixing which atomic propositions hold at each world. An agent *knows* φ at a world `w` iff φ holds at every world the agent considers accessible from `w`. Equivalence (reflexive, symmetric, transitive) accessibility relations give the standard S5 logic of knowledge.
