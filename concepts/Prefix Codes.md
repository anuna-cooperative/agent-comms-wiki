# Prefix Codes

Codes in which no codeword is a proper prefix of another, enabling unambiguous decoding from a stream without separators and satisfying Kraft's inequality. Used to define prefix Kolmogorov complexity K(x), which underpins the algorithmic-statistics/MDL apparatus.
