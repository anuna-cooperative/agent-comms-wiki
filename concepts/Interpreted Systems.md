# Interpreted Systems

The Fagin–Halpern–Moses–Vardi model for reasoning about knowledge in *distributed* and *multi-agent* systems. A global state is a tuple of each agent's local state plus the environment; a *run* is a function from time to global states, and a *system* is a set of runs. Two global states are indistinguishable to agent *i* exactly when *i*'s local state is the same in both — this indistinguishability relation induces the [[Kripke Structure|Kripke]] accessibility used to interpret knowledge formulae. Interpreted systems ground knowledge in what an agent can actually observe, connecting [[Epistemic Logic]] to protocol analysis.
