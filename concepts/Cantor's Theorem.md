# Cantor's Theorem

For every set *M*, the power set 𝒫(*M*) has strictly greater [[Cardinality]] than *M*: there is no surjection from *M* onto its subsets. Proved by [[Diagonalization]]. The immediate consequence is that there is no largest cardinal and no universal set of maximal size — the transfinite cardinalities ascend without bound.
