# Frame Logic

F-Logic (Kifer, Lausen, Wu): a logic combining the declarative semantics of first-order logic with the object-oriented, frame-based structuring of knowledge — classes, attributes, inheritance, methods — as first-class terms. A foundational formalism for ontology languages surveyed in the Handbook on Ontologies.
