# OpenID4VP

**OpenID for Verifiable Presentations** — an OpenID Foundation protocol that extends the OAuth 2.0 / OpenID Connect family to request and deliver [[Verifiable Presentation|verifiable presentations]] of [[Verifiable Credential|credentials]] held in a user's wallet. A verifier sends a *presentation request* describing the required claims; the wallet returns a cryptographically verifiable presentation derived from the holder's credentials, supporting selective disclosure. It is one of the credential-exchange standards that [[x401 - HTTP Proof Requirement Protocol|x401]] composes with rather than replacing, supplying the actual presentation mechanism behind an HTTP proof requirement.
