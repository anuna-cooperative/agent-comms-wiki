# Privacy Reasoning

An agent's capacity to judge what information may appropriately flow to whom in a given context — not as a fixed rule set but as situated, norm-sensitive inference. "Privacy Reasoning in Ambiguous Contexts" studies how LLM agents make [[Disclosure Decisions]] when the context underdetermines the norm, drawing on [[Contextual Integrity]] (appropriateness defined by context-relative information-flow norms) rather than blanket secrecy.
