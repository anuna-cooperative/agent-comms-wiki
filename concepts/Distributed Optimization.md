# Distributed Optimization

The problem of jointly minimising a global objective (often a sum of per-agent local costs) using only local computation and peer-to-peer messages. Standard tools include potential games, gradient consensus, and best-response dynamics; the class is sensitive to communication failures in ways formalised by Gopalakrishnan et al.
