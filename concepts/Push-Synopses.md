# Push-Synopses

A generalisation of Push-Sum (Kempe, Dobra, Gehrke) that computes arbitrary linear synopses — random projections, sketches, histograms — by having each node maintain a weighted synopsis and halve on each exchange, preserving the mass-conservation invariant. Subsumes averaging, counting, and moment computation.
