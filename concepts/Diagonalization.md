# Diagonalization

A self-referential proof technique: given any purported complete enumeration of objects, construct a new object that disagrees with the *n*-th listed object at "position *n*", so the new object cannot appear anywhere in the list — a contradiction. Introduced by Cantor (1891) to prove that the infinite binary sequences (equivalently 𝒫(ℕ), the reals) cannot be enumerated, establishing distinct sizes of infinity and [[Cantor's Theorem]]. The same structural move recurs throughout the limitative results of logic and computation: Gödel's incompleteness construction (a sentence asserting its own unprovability), Turing's halting problem and Church's undecidability of the Entscheidungsproblem (a program that contradicts any decider applied to itself), [[Russell's Paradox]], and Tarski's undefinability of truth. Diagonalization is thus the common engine behind "no system can completely capture / enumerate / decide its own scope."
