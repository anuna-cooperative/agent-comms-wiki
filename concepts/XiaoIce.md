# XiaoIce

Microsoft's social chatbot (launched 2014 in China; 100M+ users) built around the CPS metric, with an architecture comprising chat manager, core chat (retrieval + generation + personality + ethics filter), visual awareness, and domain skills. The flagship example of EQ-centric conversational design.
