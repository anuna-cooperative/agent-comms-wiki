# Agent Discovery

The problem and mechanisms by which agents in an open network locate peers with required capabilities. Approaches range from centralised facilitators/matchmakers (OAA, KQML) through capability descriptors (A2A Agent Cards) to decentralised identifier systems (ANP's DIDs with JSON-LD).
