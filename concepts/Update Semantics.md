# Update Semantics

The view that the meaning of a sentence is its *context-change potential* — a function from an input information state to an updated one — rather than a static truth condition ("you know the meaning of a sentence if you know the change it brings about in the information state of anyone who accepts it"). In Stalnaker's version an assertion updates the [[Common Ground]] / [[Context Set]] by eliminating worlds incompatible with it. Closely allied to [[Dynamic Semantics]].
