# ALIGN

**Agentic Linguistic Gossip Network** — the framework introduced in [[Talk Judge Cooperate - Gossip-Driven Indirect Reciprocity in Self-Interested LLM Agents]] (Zhu et al., ICML 2026) in which a population of decentralized, self-interested [[LLM Agents]] sustains cooperation by exchanging open-ended natural-language *gossip* about one another and using it to decide whom to help. Each agent is both a **talker** (emitting hierarchically-toned reputational signals) and a **judge** (assessing peers' trustworthiness from received gossip), realising [[Indirect Reciprocity]] over a linguistic [[Gossip Protocols|gossip network]] with no central reputation authority; the emergent norm identifies and excludes persistent defectors.
