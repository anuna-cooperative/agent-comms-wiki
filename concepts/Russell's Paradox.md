# Russell's Paradox

The contradiction (Russell 1901) that sank naive set theory: let *R* = {*x* : *x* ∉ *x*}, the set of all sets that are not members of themselves; then *R* ∈ *R* iff *R* ∉ *R*. It shows that unrestricted comprehension — "every property determines a set" — is inconsistent. [[Untersuchungen über die Grundlagen der Mengenlehre I|Zermelo's]] Axiom of Separation is the standard cure: predicates may only carve subsets out of already-given sets, so the universal set *R* is never formed.
