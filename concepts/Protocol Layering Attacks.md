# Protocol Layering Attacks

Attacks that exploit semantic gaps between the layers of a protocol stack — e.g. between ASN.1 parsing, X.509 interpretation, and application-level name comparison — so that inputs that are benign at one layer become malicious at another. The structural attack class exposed across the PKI Layer Cake.
