# ASCENS

Autonomic Service-Component Ensembles — an EU FP7 project that proposed a science-of-ensembles framework in which large-scale collectives of components achieve goals through self-adaptation, self-expression, and self-awareness backed by formal language foundations (SCEL) and runtime support.
