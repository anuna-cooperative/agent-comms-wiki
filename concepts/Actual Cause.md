# Actual Cause

Token-level ("this particular event caused that one") as opposed to type-level causation. Pearl formalises it with structural counterfactuals: roughly, X = x is an actual cause of Y = y if, holding fixed some context, varying X would have changed Y (the Halpern–Pearl definition, refining the but-for test to handle pre-emption and overdetermination). The apex of the [[Ladder of Causation]].
