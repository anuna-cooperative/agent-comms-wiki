# Power Set

The set 𝒫(*M*) of all subsets of a set *M*. Its existence is guaranteed by the power-set axiom of [[ZFC]]. By [[Cantor's Theorem]] it always has strictly greater [[Cardinality]] than *M*, so iterating it generates ever-larger infinities and, transfinitely, the [[Cumulative Hierarchy]].
