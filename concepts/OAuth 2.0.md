# OAuth 2.0

The IETF authorization framework (RFC 6749) that lets a client obtain delegated, scoped access to a protected resource on behalf of a resource owner without handling the owner's credentials. An authorization server issues *access tokens* — typically short-lived bearer tokens — that the client presents to the resource server; scopes bound the granted authority. OAuth 2.0 is an *authorization* (not authentication) protocol, and is one of the existing standards that [[x401 - HTTP Proof Requirement Protocol|x401]] composes with — its token-exchange leg can trade a verified credential proof for a reusable OAuth-style access token.
