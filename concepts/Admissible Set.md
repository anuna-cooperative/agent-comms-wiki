## Admissible Set

In an [[Argumentation Framework]], a set `S` of arguments that is conflict-free (no `a, b ∈ S` with `a` attacking `b`) and *defends each of its members*: every attacker on any `a ∈ S` is itself attacked by some member of `S`. Admissibility is the minimum coherence condition; the various Dung extensions (complete, [[Grounded Extension|grounded]], [[Preferred Extension|preferred]], [[Stable Extension|stable]]) all strengthen it.
